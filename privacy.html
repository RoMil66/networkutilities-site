Datenschutzerklärung

Network ScanKit — Zuletzt aktualisiert: 11. Mai 2026

Kurzfassung

Network ScanKit führt alle Netzwerkabfragen direkt auf Ihrem Gerät aus. Es werden keine personenbezogenen Daten erhoben, gespeichert, übertragen oder verarbeitet.

Welche Daten verarbeitet die App?

Die App verarbeitet ausschließlich die vom Benutzer eingegebenen Netzwerkadressen (Hostnamen, IP-Adressen, Domains, MAC-Adressen, URLs). Diese Eingaben werden nur für die jeweilige Abfrage verwendet und nicht gespeichert.

Netzwerkzugriff

Die App benötigt Netzwerkzugriff, um ihre Kernfunktionen auszuführen (z.B. Ping, DNS-Abfragen, Port-Scans, Whois, HTTP-Header-Abfragen). Alle Abfragen gehen direkt an die vom Benutzer angegebenen Ziele — es gibt keinen zwischengeschalteten Server des Entwicklers.

Die einzige externe Schnittstelle ist der IP-Geolocation-Dienst (ip-api.com), der nur bei expliziter Nutzung des „IP Geolocation"-Tools kontaktiert wird.

Tracking & Analyse

Die App enthält kein Tracking, keine Cookies, keine Analyse-SDKs, keine Werbung und keine Telemetrie.

Datenspeicherung

Es werden keine Abfrageergebnisse, Verlaufsdaten, Caches oder Logfiles dauerhaft gespeichert. Alle Daten existieren nur während der aktiven Nutzung im Arbeitsspeicher.

Drittanbieter

  • ip-api.com — Wird nur beim IP-Geolocation-Tool verwendet. Unterliegt deren Datenschutzbestimmungen.

Darüber hinaus werden keine Drittanbieter-Dienste oder SDKs eingesetzt.

Privacy Policy

Network ScanKit — Last updated: May 11, 2026

Summary

Network ScanKit performs all network queries directly on your device. No personal data is collected, stored, transmitted, or processed.

What data does the app process?

The app only processes network addresses entered by the user (hostnames, IP addresses, domains, MAC addresses, URLs). These inputs are used solely for the respective query and are not stored.

Network access

The app requires network access to perform its core functions (e.g. ping, DNS queries, port scans, whois, HTTP header requests). All queries go directly to the user-specified targets — there is no intermediary server operated by the developer.

The only external API is the IP geolocation service (ip-api.com), which is only contacted when the user explicitly uses the "IP Geolocation" tool.

Tracking & Analytics

The app contains no tracking, no cookies, no analytics SDKs, no advertising, and no telemetry.

Data storage

No query results, history data, caches, or log files are permanently stored. All data exists only in memory during active use.

Third parties

  • ip-api.com — Used only by the IP Geolocation tool. Subject to their privacy policy.

No other third-party services or SDKs are used.
